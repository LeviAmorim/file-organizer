# File Organizer

Python script that automatically organizes files into folders based on file type.

## Features
- Organizes images
- Organizes documents
- Organizes music files
- Organizes videos
- Organizes code files

## Technologies
- Python

## Goal
Practice automation and file management using Python.

## How to Use
1. Change the folder path in `organizer.py`
2. Run the script
3. Files will be automatically organized
